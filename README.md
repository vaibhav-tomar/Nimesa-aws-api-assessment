Steps to run the application
Springboot version - 3.3.1
Hibernate- 6.5.2

Prerequisite - Any IDE  intellij or eclipse, Java 17, Mysql 8
1. Clone the staging branch code from the repository.
2. Enable annotation processing for lombok and run **mvn clean package** or **mvn clean install** in project directory.
3. Replace database username and password with your username and password, and create a database with name **nimesa_aws**
3. Replace aws access key and secret key with actual credentials in application.properties file.
4. Build your project and run the server.
